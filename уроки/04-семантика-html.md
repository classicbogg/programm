## Главное

- Семантика = тег описывает **роль** блока.
- `<main>` — один на страницу.
- `id` уникален; `class` — можно повторять.

**Emmet для шапки:** вставляй теги по очереди внутри `body`:

1. `header` → Tab
2. внутри — `h1` → Tab
3. `nav` → Tab → `ul` → Tab → `li` → Tab → `a` → Tab (повтори `li` + `a` для второй ссылки)

Атрибуты и тексты дописывай вручную.

### Плохо / Хорошо

```html
<div class="header">
  <div class="nav">...</div>
</div>

<header>
  <nav>...</nav>
</header>
```

## Типичные ошибки урока

1. На странице больше одного `main` — оставь один главный блок контента.
2. Всё завернуто в `div` вместо `header`, `nav`, `section`, `footer`.
3. Классы названы по цвету (`.red`, `.blue`) вместо роли блока (`.site-header`, `.about`).
