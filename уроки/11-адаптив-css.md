## Главное

**Mobile first** — база для узкого экрана, в `@media (min-width: 768px)` расширяешь.

```css
:root {
  --color-primary: #2563eb;
  --color-text: #1f2937;
  --color-bg: #f9fafb;
}

@media (min-width: 768px) {
  .nav { flex-direction: row; }
  .cards { flex-direction: row; }
}
```

Проверка: DevTools → **Ctrl+Shift+M**.

## Типичные ошибки урока

1. Нет `meta viewport` в `<head>` — мобильная версия не масштабируется.
2. Пишешь стили только для десктопа — начни с базы для узкого экрана (mobile first).
3. Горизонтальный скролл на 320px — проверь ширины, картинки и длинные слова.

## Микро-ревью (необязательно, но полезно)

Перед итоговым проектом можно сделать короткий контроль:

1. Снова открой страницу на `320px` и `1200px`
2. Проверь, что у тебя не ломаются:
   - меню
   - карточки
   - блок контактов/форма
3. Быстро проговори вслух:
   - что у тебя задаётся в базе (mobile first)
   - что меняется в `@media (min-width: 768px)`

Это не обязательно, но помогает зайти в урок 12 без паники.
