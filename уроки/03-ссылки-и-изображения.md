## Главное

- Ссылка → **`href`**, картинка → **`src`**.
- У каждой картинки — осмысленный **`alt`**.
- Текст ссылки должен быть понятен без контекста.

**Emmet:** вставляй теги по одному: `a` → Tab (допиши `href` и текст), `img` → Tab (допиши `src` и `alt`).

### Минимум доступности (запомни на весь курс)

| Элемент | Правило |
|---------|---------|
| `img` | всегда `alt` |
| `a` | текст «Мой GitHub», не «тут» или «клик» |
| Формы (урок 5) | у каждого поля будет `label` |

### Плохо / Хорошо

```html
<a href="https://github.com/user">тут</a>
<img src="photo.jpg">

<a href="https://github.com/user">Мой GitHub</a>
<img src="images/photo.jpg" alt="Фото профиля">
```

## Типичные ошибки урока

1. Путаешь `href` (ссылка) и `src` (картинка) — стили не помогут, проверь атрибут.
2. Картинка не грузится из-за пути: файл в `images/`, а в HTML указано без папки или с опечаткой.
3. Текст ссылки «тут» / «клик» — замени на понятное описание («Мой GitHub», «Написать на почту»).
