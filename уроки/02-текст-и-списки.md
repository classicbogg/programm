## Главное

- Один `<h1>` на странице.
- `h2` → `h3` по убыванию, без пропусков уровней.
- **Emmet:** вставляй теги по одному через подсказки: `h1` → Tab, `p` → Tab, `ul` → Tab, `li` → Tab.

### Плохо / Хорошо

```html
<!-- плохо: два h1 -->
<h1>Иван</h1>
<h1>Программист</h1>

<!-- хорошо -->
<h1>Иван</h1>
<p>Начинающий программист</p>
```

```html
<!-- плохо: li без ul -->
<li>Готовлю ужин</li>

<!-- хорошо -->
<ul>
  <li>Готовлю ужин</li>
</ul>
```
