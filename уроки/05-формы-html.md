## Главное

- `label for` = `input id` — обязательная связка (доступность).
- `placeholder` — подсказка, **не замена** label.
- Таблицы — для расписаний и данных, не для вёрстки страницы.
- Для кнопок в форме указывай `type` явно:
  - `type="submit"` — отправка формы
  - `type="button"` — обычная кнопка без отправки

**Emmet для формы:** вставляй теги по одному:

1. `form` → Tab
2. `label` → Tab, `input` → Tab (повтори для каждого поля)
3. `textarea` → Tab, `select` → Tab, `button` → Tab

`for`, `id`, `type`, `required` и тексты заполняй вручную.

### Плохо / Хорошо

```html
<input type="email" placeholder="Email">

<label for="email">Email</label>
<input type="email" id="email" name="email" required>
```

## Типичные ошибки урока

1. `placeholder` вместо `label` — у каждого поля должна быть связка `label` + `id`.
2. `for` у `label` не совпадает с `id` у `input` — клик по подписи не фокусирует поле.
3. Таблица используется для вёрстки страницы, а не для данных (расписание, цены).
