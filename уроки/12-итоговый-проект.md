- Собрать полный лендинг из обязательных секций

- Подготовить проект к защите (объяснить структуру и стили)

## Главное

- Итоговый проект собирает все навыки курса в одном лендинге.
- Важнее стабильная структура и адаптив, чем “красивости”.
- Ты должен уметь объяснить каждый блок HTML/CSS в своём проекте.
- Используй реальный текст: понятные заголовки, нормальные CTA, осмысленные ссылки (не "кликни тут").

## Практика

Сверстай лендинг на тему на выбор: курсы, кофейня, услуги, проект группы.

Если хочешь идти без перегруза: сначала сделай обязательную структуру и адаптив, потом улучшай тексты и визуал.

### Структура лендинга

| Блок | Содержание |
|------|------------|
| Header | Логотип + навигация (якоря `#about`) |
| Hero | Заголовок, подзаголовок, кнопка |
| О продукте | Текст + картинка (flex) |
| Преимущества | 3–4 карточки |
| Контакты | Форма или ссылки |
| Footer | Копирайт |

### Два примера тем с готовой структурой текста

#### Тема 1: Курсы Front-end для школьников

- **Header (меню):** Главная / Программа / Отзывы / Контакты
- **Hero:**
  - Заголовок: `Научись делать сайты за 6 недель`
  - Подзаголовок: `Практический курс HTML + CSS для новичков`
  - Кнопка: `Записаться на вводный урок`
- **О продукте:**
  - Заголовок: `Как проходит обучение`
  - Текст: `Каждую неделю — короткая теория, много практики и мини-проект. Учимся через реальные задачи, а не через сухие определения.`
- **Преимущества (карточки):**
  - `Понятная программа с нуля`
  - `Поддержка наставника`
  - `Итоговый проект в портфолио`
- **Контакты:**
  - Заголовок: `Оставь заявку`
  - Подпись: `Ответим в течение дня и расскажем, с чего начать`
- **Footer:** `© 2026 Front-end Start`

#### Тема 2: Мини-кофейня в центре города

- **Header (меню):** О нас / Меню / Преимущества / Контакты
- **Hero:**
  - Заголовок: `Кофе, в который влюбляешься с первого глотка`
  - Подзаголовок: `Свежая обжарка, уютная атмосфера и десерты каждый день`
  - Кнопка: `Посмотреть меню`
- **О продукте:**
  - Заголовок: `Почему к нам возвращаются`
  - Текст: `Мы готовим кофе из зерна свежей обжарки и создаём место, где приятно работать, встречаться и отдыхать.`
- **Преимущества (карточки):**
  - `Зерно свежей обжарки`
  - `Быстрое обслуживание`
  - `Уютный зал и Wi-Fi`
- **Контакты:**
  - Заголовок: `Найди нас`
  - Подпись: `Центр города, работаем ежедневно с 8:00 до 22:00`
- **Footer:** `© 2026 Coffee Point`

Используй любой вариант как основу и адаптируй под свой проект.

### Требования

- [ ] Семантический HTML
- [ ] Отдельный `styles.css`, осмысленные классы
- [ ] Flexbox: шапка, hero, карточки, блок «о нас»
- [ ] `@media (min-width: 768px)`
- [ ] CSS-переменные
- [ ] alt, label, :focus
- [ ] Без JavaScript

### Порядок работы

1. HTML всех секций (Emmet: `!` + секции через `section#id>h2+p`).
2. CSS: переменные, шапка, hero.
3. Карточки и блок с картинкой.
4. Адаптив, реальный текст.
5. Валидатор + проверка 320px / 1200px.

### Критерии «сдал»

| | 0 | 1 | 2 |
|---|--|--|--|
| HTML | div везде | частично | семантика |
| CSS | хаос | работает | классы, переменные |
| Flexbox | нет | шапка | шапка + карточки + about |
| Адаптив | ломается | мелкие баги | 320–1200px |
| a11y | нет alt | частично | alt, label, focus |

**8+ баллов** — курс пройден.
