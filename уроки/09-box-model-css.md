## Главное

**margin** → **border** → **padding** → **content**.

```css
main {
  max-width: 800px;
  margin: 0 auto;
  padding: 0 16px;
}

img {
  max-width: 100%;
  height: auto;
}
```

**CSS:** свойства пиши обычным текстом (`padding`, `margin`, `border`, `box-sizing`).

## Типичные ошибки урока

1. Забыл `box-sizing: border-box` — блоки «прыгают» при добавлении padding.
2. Картинка шире экрана — добавь `max-width: 100%` для `img`.
3. Контент прилипает к краям — нужны `padding` у секций или контейнера.
