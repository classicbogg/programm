## Главное

- HTML описывает **содержимое и структуру**, не внешний вид.
- `<!DOCTYPE html>` — всегда первая строка.
- `<head>` — служебная информация; `<body>` — видимая часть.

**Emmet:** каркас — `!` + Tab; отдельные теги — через подсказки (`p`, `meta`, `title`). CSS — `link:css` + Tab.
