## Emmet — быстрое написание HTML

Emmet разворачивает **сокращения** в готовый код.

В этом курсе используем только такой метод:

`набрал → выбрал в списке → посмотрел описание → подтвердил Tab → проверил результат`

Так ты одновременно учишь теги и Emmet, без стресса.

Если мало времени, для старта достаточно трёх команд:

1. `!` — базовый каркас страницы
2. `ul>li*3` — быстрый список
3. `link:css` — подключение файла стилей

Остальные примеры в этом уроке — для ускорения работы, можно добрать позже.

### Базовый шаблон страницы

В пустом `index.html` напиши:

```
!
```

Нажми **Tab** — получишь полный каркас HTML5 (doctype, head, meta, body).  
То же самое: `html:5` + Tab.

### Теги и классы

| Пишешь | Получаешь |
|--------|-----------|
| `p` + Tab | `<p></p>` |
| `h1` + Tab | `<h1></h1>` |
| `div.container` + Tab | `<div class="container"></div>` |
| `header.site-header` + Tab | `<header class="site-header"></header>` |
| `p.text.highlight` + Tab | `<p class="text highlight"></p>` |
| `div#about` + Tab | `<div id="about"></div>` |

### Вложенность: символ `>`

| Пишешь | Получаешь |
|--------|-----------|
| `ul>li` + Tab | список с одним пунктом |
| `ul>li*5` + Tab | список из **5** пунктов |
| `nav>ul>li*3>a` + Tab | навигация: 3 ссылки в списке |
| `header>.logo+nav` + Tab | шапка с логотипом и nav рядом |

### Атрибуты: `[ ]`

| Пишешь | Получаешь |
|--------|-----------|
| `a[href="https://github.com"]` + Tab | ссылка с href |
| `img[src="photo.jpg"][alt="Фото"]` + Tab | картинка с src и alt |
| `input[type=email][required]` + Tab | поле email |

### Текст внутри: `{ }`

```
a[href="#"]{Главная}
```

→ `<a href="#">Главная</a>`

### Соседние элементы: `+`

```
h1+p+p
```

→ заголовок и два абзаца подряд.

### Группировка: `( )`

```
header>(.logo{Имя}+nav>ul>li*2>a{Пункт $})
```

Разверни и посмотри — потом упростишь под себя.

## Emmet для CSS

В файле `.css` тоже работает Tab-развёртка:

| Пишешь | Свойство |
|--------|----------|
| `m10` + Tab | `margin: 10px;` |
| `mt20` + Tab | `margin-top: 20px;` |
| `p16` + Tab | `padding: 16px;` |
| `df` + Tab | `display: flex;` |
| `jcc` + Tab | `justify-content: center;` |
| `aic` + Tab | `align-items: center;` |
| `fdc` + Tab | `flex-direction: column;` |
| `bgc` + Tab | `background-color: #`; |
| `w100p` + Tab | `width: 100%;` |
| `bsbb` + Tab | `box-sizing: border-box;` |

## Практика Emmet

1. Создай `index.html`, набери `!` + Tab — сохрани.
2. В `<body>` набери `h1{Моя визитка}+p{Привет}+ul>li*3` + Tab.
3. Открой через Live Server — страница должна отображаться.
4. Создай `css/styles.css`, напиши `body{m0}` + Tab → `margin: 0;`.
5. Подключи CSS в `<head>`: набери `link:css` + Tab и поправь путь на `css/styles.css`.
