## Как открыть сайт в браузере

**Способ 1 (удобный):** правый клик по `index.html` → **Open with Live Server**.  
Адрес будет вида `http://127.0.0.1:5500/...`

**Способ 2:** двойной клик по `index.html` — откроется как `file:///...`

Для курса лучше **Live Server** — меньше сюрпризов с путями к файлам.

### Live Server: как пользоваться (пошагово)

1. Открой папку `my-site` в VS Code (именно папку, не только файл).
2. Открой `index.html`.
3. Нажми правой кнопкой по файлу `index.html` в проводнике слева.
4. Выбери **Open with Live Server**.
5. Проверь в браузере адрес вида `http://127.0.0.1:5500/...`.
6. Измени текст в `index.html` и нажми `Ctrl+S` — страница должна обновиться.

Если пункта **Open with Live Server** нет:
- проверь, что расширение **Live Server** установлено;
- нажми `Ctrl+Shift+P` → введи `Live Server: Open with Live Server`.

## Emmet — вставка тегов через подсказки

Emmet встроен в VS Code. В курсе два уровня:

1. **Базовые сокращения** — знаем и используем: `!`, `link:css`
2. **Отдельные теги** — вставляем через подсказки: `p` → Tab → `<p></p>`
3. **Сложные сокращения** — пока не используем: `ul>li*3`, `header>nav`, `h1{текст}` и т.п.

**Метод (запомни на весь курс):**

`набрал → выбрал в списке → посмотрел описание → Tab → проверил результат`

### Базовые сокращения (обязательно)

| Набираешь | После Tab получаешь |
|-----------|---------------------|
| `!` | полный каркас HTML5 (`doctype`, `head`, `meta`, `body`) |
| `link:css` | `<link rel="stylesheet" href="style.css">` — поправь путь на `css/styles.css` |

То же самое для каркаса: `html:5` + Tab.

### Отдельные теги через подсказки

| Набираешь | После Tab получаешь |
|-----------|---------------------|
| `p` | `<p></p>` |
| `h1` | `<h1></h1>` |
| `h2` | `<h2></h2>` |
| `ul` | `<ul></ul>` |
| `li` | `<li></li>` |
| `a` | `<a href=""></a>` |
| `img` | `<img src="" alt="">` |
| `header` | `<header></header>` |
| `nav` | `<nav></nav>` |
| `main` | `<main></main>` |
| `section` | `<section></section>` |
| `footer` | `<footer></footer>` |

Атрибуты (`href`, `src`, `alt`, `class`) дописываешь **вручную** внутри тега.

Сложные комбинации (`ul>li*5`, `header>h1+nav`) **не используем** — собирай структуру тегами по одному.

### Каркас страницы

В пустом `index.html` набери `!` + Tab — получишь готовый каркас.  
Альтернатива: скопировать файл из [`шаблон/`](../шаблон/).

### CSS

Свойства в `styles.css` пишешь **обычным текстом**, без Emmet-сокращений:

```css
body {
  margin: 0;
  padding: 16px;
}
```
