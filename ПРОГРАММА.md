# Программа курса

**13 шагов:** урок 0 (настройка) + 6 HTML + 6 CSS (включая итоговый проект).

---

## Легенда: как читать таблицы уроков

- `Урок` — ссылка на файл занятия
- `Ключевые темы` — что изучаешь в этом уроке
- `На выходе` — что должно быть готово после урока
- `Подсказка` — быстрый практический совет, чтобы не застрять

Мини-порядок внутри каждого урока:
`Прочитать → Практика → Сдать → Переход → Самопроверка`

Если мало времени: в каждом уроке ориентируйся на блок `Минимум урока`.

## CSS — уроки 7–12

| Урок | Тема | Ключевые темы | На выходе | Подсказка |
|------|------|----------------|-----------|-----------|
| [7](./lessons/07-основы-css.md) | Основы CSS | link, селекторы, имена классов | Подключен `styles.css`, базовые стили работают | Используй осмысленные классы: `.site-header`, не `.div1` |

| [11](./lessons/11-адаптив-css.md) | Адаптив | @media, rem, переменные | Адаптивный черновик лендинга | Проверяй минимум `320px` и `1200px` |
