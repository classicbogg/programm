## Десктоп (широкий экран)

```
┌─────────────────────────────────────────────┐
│  HEADER                                     │
│  [Логотип]              Главная  О нас  ... │
├─────────────────────────────────────────────┤
│                    HERO                     │
│              Большой заголовок              │
│              Подзаголовок                   │
│              [ Кнопка CTA ]                 │
├─────────────────────────────────────────────┤
│  О ПРОДУКТЕ (#about)                        │
│  ┌──────────────┐  ┌──────────────────────┐ │
│  │   картинка   │  │  текст, 2–3 абзаца   │ │
│  └──────────────┘  └──────────────────────┘ │
├─────────────────────────────────────────────┤
│  ПРЕИМУЩЕСТВА                               │
│  ┌────────┐  ┌────────┐  ┌────────┐         │
│  │ карта 1│  │ карта 2│  │ карта 3│         │
│  └────────┘  └────────┘  └────────┘         │
├─────────────────────────────────────────────┤
│  КОНТАКТЫ (#contacts)                       │
│  форма или email / ссылки                   │
├─────────────────────────────────────────────┤
│  FOOTER — © 2026                            │
└─────────────────────────────────────────────┘
```

## С чего начать код

1. HTML всех секций с заглушками текста.
2. CSS: сброс, переменные, шапка.
3. Hero + карточки (flex).
4. Блок «о продукте» (flex row на десктопе).
5. `@media` для мобилы.
6. Реальный текст, favicon.
