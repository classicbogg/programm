# Практика к уроку 0: настройка и Emmet

## Навигация

[ЧЕКЛИСТ](./ЧЕКЛИСТ.md) · [Урок 01 →](./урок-01-практика.md)

---

**Ориентир по времени:** 45–60 минут.

**Цель:** подготовить рабочее окружение и первый черновик проекта.

**Контекст:** ты начинаешь учебный сайт `my-site`. В этом уроке — только старт: файлы, Emmet, Live Server и первое подключение CSS.

## База

1. Создай папку `my-site` и открой её в редакторе как проект (File → Open Folder).
2. Создай структуру:
   - `index.html`
   - `css/styles.css`
   - `images/`
3. Установи и проверь расширение **Live Server**.
4. Создай пустой `index.html`, набери `!` + Tab — сохрани.
5. Открой страницу через **Open with Live Server**.

## Средний

1. В `<body>` вставляй теги **по одному** через подсказки: `h1` → Tab, `p` → Tab.
2. Добавь навигацию: `nav` → `ul` → три раза `li` → в каждом `a` → Tab.
3. Замени тексты ссылок и `href` вручную.
4. В `<head>` набери `link:css` + Tab, поправь путь на `css/styles.css`.
5. В `css/styles.css` напиши обычным текстом:
   ```css
   body {
     margin: 0;
     padding: 16px;
   }
   ```
6. Сохрани оба файла и проверь автообновление после `Ctrl+S`.

## Продвинутый

1. Добавь ещё 3 тега через подсказки (`header`, `footer`, `section`) и заполни содержимое.
2. Проверь метод из урока: `набрал → выбрал в списке → посмотрел описание → Tab → проверил результат`.
3. Открой DevTools (F12) → **Elements** и найди в дереве свой `header` и `nav`.
4. Запиши в комментарий в `index.html`, чем Live Server удобнее двойного клика по файлу.

## Самопроверка

- [ ] Папка проекта открыта в редакторе целиком
- [ ] Live Server запускается без ошибок
- [ ] Развернул каркас через `!` + Tab
- [ ] Теги вставляются через подсказки (`p`, `h1`, `ul`, `li`, `a`)
- [ ] CSS подключён через `link:css` + Tab
