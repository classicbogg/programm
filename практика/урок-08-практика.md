**Ориентир по времени:** 45–55 минут.

**Цель:** сделать текст читаемым и настроить состояния интерактивных элементов.

**Контекст:** сайт уже оформлен базово. Теперь доводишь типографику и визуальную иерархию.

## База

1. Настрой размеры заголовков (`h1`, `h2`, `h3`) и базовый размер текста.
2. Задай удобный `line-height` для абзацев.
3. Выбери 2-3 основных цвета (текст, фон, акцент).
4. Примени цвета к ключевым блокам сайта.

## Средний

1. Подключи Google Fonts (или используй аккуратный системный стек).
2. Настрой контраст: текст должен читаться на фоне без напряжения.
3. Добавь `:hover` для ссылок меню и кнопок.
4. Добавь `:focus` для ссылок и кнопок (видимый фокус с клавиатуры).
5. Проверь длинные абзацы: нет ли «стены текста» без воздуха.

## Продвинутый

1. Сделай типографическую шкалу: размеры заголовков и отступы между блоками текста.
2. Добавь `:active` для кнопок и ссылок.
3. Проверь страницу на узкой и широкой ширине: текст не должен «ломаться».
4. Убери визуальный шум: лишние цвета, разные шрифты без причины.
5. Сравни «до/после» и оставь только улучшения, которые реально помогают читать.

## Самопроверка

- [ ] Заголовки и текст читаются легко
- [ ] Есть `:hover` и `:focus`
- [ ] Цвета согласованы по всему сайту
- [ ] Длинные тексты не выглядят перегруженно
