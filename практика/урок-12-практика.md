# Практика к уроку 12: итоговый проект

## Навигация

[← Урок 11](./урок-11-практика.md) · [ЧЕКЛИСТ](./ЧЕКЛИСТ.md)

---

**Ориентир по времени:** 2–3 часа (можно разбить на 2–3 сессии).

**Цель:** собрать и отполировать итоговый адаптивный лендинг.

**Контекст:** финальная работа курса — **отдельный проект**, не доработка визитки.

- Создай папку `landing/` рядом с `my-site/` (учебная визитка остаётся как есть).
- Черновик блоков из урока 11 можно использовать как подсказку, но лендинг собираешь **с чистого листа**.
- Примеры тем, структура и критерии оценки — в [уроке 12](../уроки/12-итоговый-проект.md).

Структура папки:

```
landing/
  index.html
  css/styles.css
  images/
```

## База

1. Выбери тему лендинга (курсы, кофейня, услуги, проект группы).
2. Создай `landing/index.html` и разверни каркас через `!` + Tab.
3. Собери HTML всех обязательных секций (теги через подсказки: `section`, `h2`, `p`, `article`):
   - Header (логотип + навигация с якорями)
   - Hero (заголовок, подзаголовок, кнопка)
   - О продукте (текст + изображение)
   - Преимущества (3–4 карточки)
   - Контакты (форма или ссылки)
   - Footer
4. Используй реальные тексты, не «Lorem ipsum».
5. Проверь якорные ссылки: клик по пункту меню ведёт к нужной секции.

## Средний

1. Создай `landing/css/styles.css` и подключи через `<link>` в `<head>`.
2. Задай CSS-переменные в `:root`, оформи шапку и hero на Flexbox.
3. Сверстай карточки и блок «о продукте» на Flexbox.
4. Добавь `@media (min-width: 768px)` и проверь `320px → 1200px` без горизонтального скролла.
5. Прогони `landing/index.html` через валидатор и исправь ошибки.

## Продвинутый

1. Проверь доступность: `alt`, `label`, `:focus`, понятные ссылки и кнопки.
2. Сделай финальный QA на ширинах 320 / 768 / 1024 / 1200.
3. Пройди [ЧЕКЛИСТ.md](./ЧЕКЛИСТ.md) полностью.
4. Сверься с критериями «сдал» из урока 12 (цель — 8+ баллов).
5. Подготовь объяснение проекта (3–5 тезисов) и сделай демо-показ наставнику.

## Самопроверка

- [ ] Проект в отдельной папке `landing/`, не смешан с `my-site/`
- [ ] Все секции лендинга на месте и с реальными текстами
- [ ] Flexbox, переменные, адаптив работают на 320–1200px
- [ ] Валидатор и чек-лист пройдены
- [ ] Могу объяснить свой проект без подсказок
