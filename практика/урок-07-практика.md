---

**Ориентир по времени:** 50–60 минут.

**Цель:** подключить внешний CSS и выстроить осмысленную систему классов.

**Контекст:** HTML готов на трёх страницах. Подключи один `css/styles.css` ко всем: `index.html`, `resume.html`, `contact.html`.

## База

1. Подключи `css/styles.css` через `<link>` в `<head>` на всех трёх страницах.
2. Добавь базовые стили:
   - сброс `margin` у `body`
   - базовый `font-family`
3. Убери подчёркивание у ссылок в меню.
4. Открой каждую страницу через Live Server и убедись, что стили применяются.
5. Проверь: `resume.html` больше не «голая» — оформление как на остальных страницах.

## Средний

1. Переименуй классы в HTML на роль блока: `site-header`, `nav`, `site-footer`, `card`.
2. Убери «мусорные» имена вроде `div1`, `red`, `block2`.
3. Добавь стили для шапки, меню и подвала.
4. Добавь фон странице и контрастный цвет ссылок.
5. Проверь, что один и тот же CSS одинаково работает на всех трёх страницах.

## Продвинутый

1. Сделай простую UI-систему: стили для заголовков, абзацев, ссылок, кнопок.
2. Используй селекторы по классу, а не длинные цепочки тегов.
3. Добавь отдельный класс для контейнера контента (`.container`).
4. Проверь в DevTools, какие правила реально применяются к элементам.
5. Удали дублирующиеся или конфликтующие CSS-правила.

## Самопроверка

- [ ] CSS подключён на `index.html`, `resume.html`, `contact.html`
- [ ] Классы названы по роли
- [ ] Базовое оформление едино на всём сайте
- [ ] Нет инлайн-стилей в HTML
