# Урок 11. CSS: адаптив и мини-сборка

**Часть:** CSS  
**Результат:** сайт на 320px и 1200px; черновик блоков для лендинга.
**Сколько примерно займёт:** 90–120 минут.

## Минимум урока

- Добавить хотя бы один `@media (min-width: 768px)`
- Проверить страницу на `320px` и `1200px`
- Убедиться, что нет горизонтального скролла

## Главное

**Mobile first** — база для узкого экрана, в `@media (min-width: 768px)` расширяешь.

```css
:root {
  --color-primary: #2563eb;
  --color-text: #1f2937;
  --color-bg: #f9fafb;
}

@media (min-width: 768px) {
  .nav { flex-direction: row; }
  .cards { flex-direction: row; }
}
```

Проверка: DevTools → **Ctrl+Shift+M**.

---

## Практика

1. Цвета в `:root`, заголовки в `rem`.
2. Мобила: меню и карточки столбиком.
3. От 768px: в ряд.
4. Проверь `index.html` и `contact.html` на 320px и 1200px.

### Если появился горизонтальный скролл

Проверь по шагам:

1. Нет ли фиксированных ширин вроде `width: 1200px`
2. Есть ли у изображений `max-width: 100%`
3. Нет ли очень длинных слов/ссылок без переноса
4. Не выходит ли `position: absolute`-элемент за экран

### Мини-сборка перед уроком 12

На главной должны быть **все типы блоков** лендинга (можно с заглушками):

- [ ] Hero
- [ ] Секция с текстом + картинкой (flex)
- [ ] 3 карточки
- [ ] Форма или блок контактов

Схема: [СХЕМА-ЛЕНДИНГА.md](../practice/СХЕМА-ЛЕНДИНГА.md).

### Обязательный мини-демо

Перед уроком 12 покажи наставнику или одногруппнику:

1. Один и тот же экран на `320px` и `1200px`
2. Hero + блок карточек + блок контактов
3. Что именно меняется в `@media (min-width: 768px)`

## Сдать после урока

- Адаптивный черновик главной страницы с ключевыми секциями
- Проверка `320px` и `1200px` пройдена без критичных поломок
- Используются `:root` переменные и минимум один media-query

## Переход к следующему уроку

Переходи к уроку 12, только если:

1. Мини-демо показан и понятен
2. Нет горизонтального скролла на 320px
3. Блоки лендинга уже собраны в черновике

---

## Микро-ревью (необязательно, но полезно)

Перед итоговым проектом можно сделать короткий контроль:

1. Снова открой страницу на `320px` и `1200px`
2. Проверь, что у тебя не ломаются:
   - меню
   - карточки
   - блок контактов/форма
3. Быстро проговори вслух:
   - что у тебя задаётся в базе (mobile first)
   - что меняется в `@media (min-width: 768px)`

Это не обязательно, но помогает зайти в урок 12 без паники.
