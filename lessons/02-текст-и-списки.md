**Часть:** HTML  
**Результат:** заголовки, абзацы, два типа списков.

## Минимум урока

- Один `h1`, минимум два абзаца
- Один `ul` и один `ol`

## Главное

- Один `<h1>` на странице.
- `h2` → `h3` по убыванию, без пропусков уровней.
- **Emmet:** `h1{Имя}+p*2` + Tab; список навыков: `ul>li*5` + Tab.

### Плохо / Хорошо

```html
<!-- плохо: два h1 -->
<h1>Иван</h1>
<h1>Программист</h1>

<!-- хорошо -->
<h1>Иван</h1>
<p>Начинающий программист</p>
```

```html
<!-- плохо: li без ul -->
<li>HTML</li>

<!-- хорошо -->
<ul>
  <li>HTML</li>
</ul>
```

---

## Практика

1. `h1` — имя (`h1{Твоё имя}` + Tab).
2. Два абзаца о себе (`p*2` + Tab, потом текст).
3. `h2` «Навыки» + `ul>li*5` + Tab.
4. `h2` «План обучения» + `ol>li*3` + Tab.

### Дополнительно

Вложенный список: `ul>li{HTML}+li{CSS}+li{Планы:}>ul>li*2` + Tab — разберись, что получилось.
