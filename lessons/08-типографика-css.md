# Урок 8. CSS: цвета, типографика и шрифты

**Часть:** CSS  
**Результат:** единый стиль текста, hover/focus, подключён Google Font (по желанию).
**Сколько примерно займёт:** 60–90 минут.

## Минимум урока

- Выставить читаемую типографику
- Настроить `:hover` и `:focus` для ссылок/кнопок
- Ограничиться 2-3 основными цветами

## Главное

- 2–3 цвета на весь сайт.
- `line-height: 1.5–1.6` для абзацев.
- `:focus` — видимая обводка при Tab по ссылкам (доступность).

```css
a:focus {
  outline: 2px solid #2563eb;
  outline-offset: 2px;
}
```

**Emmet в CSS:** `c#222` + Tab, `lh16` + Tab, `ffs` + Tab.

---

## Google Fonts (5 минут)

1. Открой [fonts.google.com](https://fonts.google.com/), выбери шрифт (например **Inter**).
2. Скопируй `<link>` в `<head>`.
3. В CSS:

```css
body {
  font-family: "Inter", system-ui, sans-serif;
}
```

`system-ui` — запасной шрифт, если Google не загрузится.

---

## Практика

1. Палитра: фон, текст, акцент для ссылок и кнопки.
2. Стили для `h1`, `h2`, `p`, ссылок.
3. `:hover` и `:focus` на ссылках меню.
4. Стилизуй кнопку формы.

## Сдать после урока

- Настроены цвета текста, фона, акцента
- Для ссылок есть `:hover`, для интерактивных элементов есть `:focus`
- Базовая типографика (font-family, font-size, line-height) выглядит читаемо
