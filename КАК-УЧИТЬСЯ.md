# Как учиться

Каждый урок: **прочитать → сделать → проверить**.

**Старт:** [урок 0](./lessons/00-настройка.md) (Emmet, Live Server) → затем урок 1.

---

## Анти-перегруз (важно)

Если чувствуешь, что информации много, делай только в таком порядке:

1. `Главное`
2. `Практика`
3. `Сдать после урока`
4. `Переход`

Блоки `Дополнительно` и расширенные шпаргалки — только после обязательного минимума.

---

## Формат урока

```
Прочитать    →  статьи на Доке
Главное      →  правила, Emmet, примеры
Практика     →  код в my-site/
Сдать после урока
Переход      →  проверь готовность к следующему уроку
Проверь себя →  3 вопроса
Самопроверка →  галочки
```

Прочитал статью — сразу повтори в редакторе.  
Используй **Emmet** (`!` + Tab, `ul>li*5` + Tab) — [ШПАРГАЛКА.md](./ШПАРГАЛКА.md).

---

## Папка проекта

Скопируй [starter/](./starter/) → переименуй в `my-site/`:

```
my-site/
  index.html
  contact.html    ← урок 5
  css/styles.css  ← урок 7
  images/
```

Один проект на весь курс.

---

## Как читать Доку

1. Только ссылки из урока, не весь раздел.
2. Достаточно прочитать до примеров.
3. Термин непонятен — кликни в статье или [глоссарий](https://doka.guide/).

---

## Если застрял

1. Блок **«Главное»** в уроке.
2. [ШПАРГАЛКА.md](./ШПАРГАЛКА.md).
3. [ЕСЛИ-ЗАСТРЯЛ.md](./ЕСЛИ-ЗАСТРЯЛ.md).
4. DevTools (F12) → Elements.
5. Покажи **файл** наставнику.

## Переход к следующему уроку

- Сайт открывается в браузере
- Блоки **Сдать после урока** и **Переход** закрыты
