# Курс Front-end для начинающих (HTML + CSS)

Учебные материалы для самостоятельного изучения вёрстки с нуля: уроки, практика, шпаргалки, итоговый лендинг.

**С чего начать после скачивания:** открой [СТАРТ.md](./СТАРТ.md) и иди по [маршруту обучения](./МАРШРУТ-ОБУЧЕНИЯ.md).

---

## Как скачать репозиторий на компьютер

### Способ 1 — через Git (рекомендуется)

Нужен [Git](https://git-scm.com/download/win). После установки открой терминал (PowerShell или «Git Bash») в папке, куда хочешь положить курс, и выполни:

```bash
git clone https://github.com/classicbogg/programm.git
```

Появится папка `programm` с материалами курса.

Открой её в VS Code: **File → Open Folder** → выбери `programm`.

### Способ 2 — без Git (архив)

1. Открой репозиторий: https://github.com/classicbogg/programm  
2. Нажми зелёную кнопку **Code** → **Download ZIP**  
3. Распакуй архив в удобное место  
4. Открой распакованную папку в VS Code

---

## Как получить обновления

Если скачивал через `git clone`, зайди в папку проекта и выполни:

```bash
cd programm
git pull
```

Так подтянутся новые уроки и правки с GitHub.
