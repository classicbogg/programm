## HTML-шаблон

```
! + Tab
```

или скопируй из [starter/index.html](./starter/index.html).

---

## Emmet — HTML

| Сокращение | Результат |
|------------|-----------|
| `!` | полная страница HTML5 |
| `tag` + Tab | тег |
| `.class` | class="class" |
| `#id` | id="id" |
| `>` | вложенность |
| `+` | соседний элемент |
| `*N` | повторить N раз |
| `{текст}` | текст внутри |
| `a[href=url]{Текст}` | ссылка |
| `img[src=][alt=]` | картинка |
| `ul>li*5` | 5 пунктов списка |
| `link:css` + Tab | тег link для CSS |

**Пример навигации:**

```
nav>ul>li*3>a{Пункт $}
```

## Именование классов

```
.site-header    — шапка сайта
.site-footer    — подвал
.nav            — навигация
.card           — карточка
.cards          — контейнер карточек
.hero           — первый экран
```

Не используй: `.div1`, `.block2`, `.red`

---

## Flexbox — типовые блоки

```css
/* шапка */
.header-inner {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

/* карточки */
.cards {
  display: flex;
  flex-wrap: wrap;
  gap: 20px;
}

.card {
  flex: 1 1 280px;
}
```

---

## Адаптив

```css
/* mobile first */
.nav { flex-direction: column; }

@media (min-width: 768px) {
  .nav { flex-direction: row; }
}
```

## CSS-переменные

```css
:root {
  --color-primary: #2563eb;
  --color-text: #1f2937;
}

button {
  background: var(--color-primary);
}
```

---

## Минимум доступности

- У `img` всегда `alt`
- У полей формы — `label` + `id`
- У ссылок — понятный текст (не «тут»)
- `:focus` на интерактивных элементах
