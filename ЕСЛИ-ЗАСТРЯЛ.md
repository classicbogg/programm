# Если застрял

Короткий справочник по частым проблемам. Сначала проверь блок **Главное** в уроке, потом — пункт ниже.

---

## 1. Страница не обновляется после правок

- Сохрани файл: `Ctrl+S`.
- Открой через **Live Server**, не двойным кликом (если работаешь с путями и CSS).
- Обнови вкладку браузера вручную (`F5`).

## 2. CSS не применяется

- Файл подключён в `<head>` через `link`?
- Путь верный: из `index.html` это `css/styles.css`, не `/css/styles.css` без причины.
- В DevTools (F12) → **Network** — файл `styles.css` загружается без ошибки 404?

## 3. Картинка не отображается

- Проверь `src`: файл лежит в `images/`, в HTML — `images/photo.jpg`.
- Учитывай регистр букв в имени файла.
- Для внешней ссылки проверь URL в браузере отдельно.

## 4. Ссылка не работает

- Для другой страницы: `contact.html`, не `contact` без расширения.
- Для якоря: `href="#about"` и на странице есть `id="about"`.
- Для `mailto:` — `href="mailto:you@mail.ru"`.

## 5. Emmet не разворачивается

- Файл сохранён как `.html` или `.css`?
- Набрал тег → выбрал подсказку в списке → **Tab**.
- Для каркаса: `!` + Tab. Для CSS-файла: `link:css` + Tab в `<head>`.

## 6. Валидатор показывает ошибки

- Незакрытый тег — найди пару открытия/закрытия.
- `li` только внутри `ul` или `ol`.
- Один `h1` на странице.
- У `img` есть `alt` (может быть пустым только для декоративных).

## 7. Горизонтальный скролл на узком экране

- У картинок: `max-width: 100%`.
- Нет ли `width: 1200px` у блоков?
- Проверь в DevTools → **Ctrl+Shift+M** на ширине 320px.

## 8. Flexbox «не работает»

- `display: flex` стоит на **родителе**, не на каждой карточке.
- Для переноса карточек: `flex-wrap: wrap` и `gap`.

## 9. Форма: клик по тексту не выделяет поле

- `label for="email"` и `input id="email"` — значения должны совпадать.
