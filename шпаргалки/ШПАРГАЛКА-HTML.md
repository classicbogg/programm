# HTML шпаргалка (расширенная)

Эта шпаргалка для учеников: открыть и быстро найти нужный тег.  
Полный справочник с деталями: [Doka HTML](https://doka.guide/html/).

---

## 1) Базовый каркас страницы

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Заголовок вкладки</title>
</head>
<body>
  ...
</body>
</html>
```

## 6) Ссылки и навигация

| Тег/атрибут | Назначение |
|------------|------------|
| `a` | ссылка |
| `href` | адрес ссылки |
| `target="_blank"` | открыть в новой вкладке |
| `rel="noopener noreferrer"` | безопасно для `_blank` |
| `id` + `href="#id"` | якорная навигация |

Примеры:

```html
<a href="https://example.com">Внешняя ссылка</a>
<a href="contact.html">Внутренняя ссылка</a>
<a href="#about">Переход к секции</a>
```

## 12) Минимум доступности (a11y)

- У каждого `img` должен быть осмысленный `alt`
- У каждого поля формы должна быть связка `label` + `for/id`
- У ссылок понятный текст (`"Подробнее о курсе"`, а не `"тут"`)
- Иерархия заголовков без хаоса (`h1` -> `h2` -> `h3`)

---

## 14) Emmet

**Базовые сокращения (используем в курсе):**

| Набираешь | После Tab |
|-----------|-----------|
| `!` | каркас HTML5 |
| `link:css` | подключение CSS-файла |

**Отдельные теги через подсказки:**

| Набираешь | После Tab |
|-----------|-----------|
| `p` | `<p></p>` |
| `h1` | `<h1></h1>` |
| `ul` | `<ul></ul>` |
| `li` | `<li></li>` |
| `a` | `<a href=""></a>` |
| `img` | `<img src="" alt="">` |
| `header` | `<header></header>` |
| `form` | `<form action=""></form>` |

Сложные сокращения (`ul>li*3`, `header>nav` и т.п.) в курсе не используем.

## 15) Что важно запомнить новичку

1. HTML = структура и смысл, CSS = внешний вид  
2. Семантика важнее “красивого div-леса”  
3. Валидатор и DevTools — обязательные инструменты
