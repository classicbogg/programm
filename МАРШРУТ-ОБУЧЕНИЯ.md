# Маршрут обучения

## Как проходить курс

Каждый урок: **прочитать -> сделать -> проверить**.

Порядок внутри урока:

`Прочитать -> Главное -> Практика -> Переход -> Проверь себя -> Самопроверка`

Правила:

- Иди строго по порядку: урок 0, 1, 2 ... 12.
- После чтения сразу повторяй пример в коде.
- Не пропускай блок `Переход к следующему уроку`.
- Emmet используй через подсказки редактора: `набрал -> выбрал в списке -> Tab`.
