# Для наставника

Краткий гид: что проверять на каждом этапе и типичные ошибки новичков.

---

## Общие правила проверки

1. Смотри **файлы**, не скриншоты.
2. Открой сайт локально (Live Server).
3. Сожми окно до **320px** — с урока 11.
4. Не давай готовый код сразу — направляй к [Доке](https://doka.guide/) и уроку.
5. Перед переходом к новому уроку проверяй блоки `Сдать после урока` и `Переход`.
6. При типовой ошибке сначала отправляй в [ЕСЛИ-ЗАСТРЯЛ.md](./ЕСЛИ-ЗАСТРЯЛ.md).

---

## Урок 0 — настройка

**Проверить:**
- [ ] Папка `my-site/` есть
- [ ] Live Server работает
- [ ] Студент развернул `!` + Tab и `ul>li*3` + Tab

**Частые проблемы:** Tab не срабатывает (нужен Enter); файл не сохранён (Ctrl+S).

---

## Уроки 1–5 — HTML

| Урок | 3 главных пункта |
|------|------------------|
| 1 | doctype, charset, страница открывается |
| 2 | один h1, ul/ol с li |
| 3 | alt у img, href у a |
| 4 | header/main/footer, main один |
| 5 | label+id, required на полях |

**Красные флаги:** несколько h1; `href` вместо `src`; форма без label; таблица для всей вёрстки.

---

## Урок 6 — итог HTML

- [ ] 2 страницы, общая шапка/подвал
- [ ] [validator.w3.org](https://validator.w3.org/) — нет критичных ошибок
- [ ] DevTools: студент показывает элемент в дереве

---

## Уроки 7–11 — CSS

| Урок | 3 главных пункта |
|------|------------------|
| 7 | внешний CSS, осмысленные классы |
| 8 | читаемый текст, :hover, :focus |
| 9 | box-sizing, main по центру |
| 10 | flex в шапке и карточках |
| 11 | 320px без горизонтального скролла, @media |

**Красные флаги:** стили в `<style>` на 200 строк; классы `.div1`; flex только на одном элементе без эффекта.

---

## Урок 12 — лендинг

Схема блоков: [practice/СХЕМА-ЛЕНДИНГА.md](./practice/СХЕМА-ЛЕНДИНГА.md)

**Зачёт (8+/10):**
- семантика
- flex: шапка + карточки
- адаптив 320–1200px
- alt, label
- отдельный CSS, переменные

**Бонус:** favicon, Google Fonts, GitHub.

---

## Типичные вопросы новичков

| Вопрос | Куда отправить |
|--------|----------------|
| «Не открывается картинка» | Путь к файлу, регистр букв |
| «Стили не применяются» | Путь в link, сохранён ли CSS |
| «Всё в столбик» | display: flex на родителе |
| «Ширина разъехалась» | box-sizing: border-box |
