## Как проходить

1. Урок 0 → урок 1 → … → урок 12 **по порядку**.
2. В каждом уроке: **Дока** → **практика** → **сдать после урока** → **переход** → **проверь себя** → **самопроверка**.
3. Используй **Emmet** (`!`, `ul>li*5`, `df` в CSS) — [ШПАРГАЛКА.md](./ШПАРГАЛКА.md).
4. Не пропускай блок **🚦 Переход к следующему уроку** — это обязательная проверка готовности.

**Итог:** адаптивный лендинг без JavaScript.

## Быстрый индекс проблем

- Не работает Emmet / Live Server → [ЕСЛИ-ЗАСТРЯЛ.md](./ЕСЛИ-ЗАСТРЯЛ.md)
- Не помню HTML-тег → [HTML шпаргалка](./cheatsheets/ШПАРГАЛКА-HTML.md)
- Не помню CSS-свойство → [CSS шпаргалка](./cheatsheets/ШПАРГАЛКА-CSS.md)
- Не понимаю, почему “ломается” блок → [ШПАРГАЛКА.md](./ШПАРГАЛКА.md) + DevTools (F12)
- Не уверен, можно ли идти дальше → открой текущий урок и закрой `📦 Сдать после урока` + `🚦 Переход`
